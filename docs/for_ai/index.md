# Карта для AI-агентов

Прочтите это перед тем, как открывать исходники — это кратчайший путь к
нужному файлу.

## Начните отсюда

1. **`src/pylzt/_MODULE_AUTO.md`** — карта пакета верхнего уровня (что
   принадлежит каждому подпакету).
2. Файл `_MODULE_AUTO.md` в каждом пакете — публичная поверхность только
   этого пакета, генерируется `docskill`, поддерживается актуальным после
   каждого изменения:
   - `src/pylzt/methods/_MODULE_AUTO.md` — `BaseMethod[T]`, контракт
     method-as-class.
   - `src/pylzt/models/_MODULE_AUTO.md`, `models/{market,forum,antipublic}/_MODULE_AUTO.md`
     — DTO ответов, базовые классы `LolzObject`/`BaseResponse`.
   - `src/pylzt/facades/_MODULE_AUTO.md` — сгенерированные namespace'ы
     `client.market`/`client.forum`/`client.antipublic` и их sync-аналоги.
   - `src/pylzt/transport/_MODULE_AUTO.md` — граница на проводе
     (`Request`/`Response`, `BaseTransport`, `HttpxSession`).
   - `src/pylzt/token_pool/_MODULE_AUTO.md`, `proxy_pool/_MODULE_AUTO.md` —
     выдача рейт-лимитов и ротация egress-прокси.
   - `src/pylzt/storage/_MODULE_AUTO.md`, `export/_MODULE_AUTO.md`,
     `sync/_MODULE_AUTO.md`, `lib/_MODULE_AUTO.md`, `enums/_MODULE_AUTO.md` —
     персистентность, экспорт по курсору, sync-движок, общие примитивы
     (batch/retry/concurrency), сгенерированные `StrEnum`.
   - `dev/codegen/_MODULE_AUTO.md` — генератор из OpenAPI-спеки в
     библиотеку.
3. Открывайте реальные исходники, только если документ отсутствует, устарел
   или вам нужен control flow, которого документ не несёт.

## Неочевидные инварианты, которые стоит знать перед редактированием

- **Method-as-class, а не написанные вручную функции.** Каждый эндпоинт —
  frozen Pydantic-модель `BaseMethod[T]`; `Client.execute(method)` —
  единственный путь выполнения, к которому делегирует каждый фасад. Не
  добавляйте второй путь построения запроса.
- **Сгенерированный код лежит плоско в библиотеке, а не в отдельном
  пакете.** Сгенерированные и написанные вручную модули сосуществуют в тех
  же директориях — сгенерированные несут заголовок
  `Generated by forge — DO NOT EDIT` и имя с префиксом `{api}_`/`{api}`,
  написанные вручную — без префикса. `install()` отказывается перезаписывать
  написанный вручную модуль (`_guard_no_clobber`).
- **Sync — не вторая реализация.** `SyncClient`/`sync/runner.py` запускает
  тот же async-движок в фоновом потоке с event loop'ом — есть ровно одна
  реализация рейт-лимита/повторов, а не по одной на sync/async вариант.
- **`GitHub Actions` сейчас не является CI-гейтом.** Им является
  `.githooks/pre-push` (ruff check, ruff format --check, mypy, pytest) — см.
  раздел "Разработка" в README.
